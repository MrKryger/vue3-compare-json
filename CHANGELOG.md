## [0.1.0] - 2026-05-19

### Added
- Initial release.
- `VueJsonCompare` component with GitHub-style unified diff for two JSON values.
- Props: `oldData`, `newData`, `showHeader`, `showLineNumbers`, `theme`, `title`, `noChangesLabel`.
- Vue plugin entry for global registration.
- CSS variables for theming, plus built-in `light` / `dark` / `auto` themes.
- TypeScript declarations bundled with the package.
